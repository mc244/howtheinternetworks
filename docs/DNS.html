Domain Name System (DNS)



The Domain Name System is basically the phonebook of the Internet. When we type in the domain name into the browser, DNS will translate the domain names to IP addresses so that browsers can load Internet resources. It is an extremely helpful tool as it helps eliminate the need for users to remember the complex IP addresses. Since there are so many unique IP addresses, it is impossible for one DNS server to be able to hold all that information. Therefore, DNS servers are connected in a distributed hierarchy, splitting the responsibilities such as by the top level domain. For example, one DNS server might be responsible for the TLD with .com while another could be repsonsible for a TLD of .org.



DNS was originally designed to be open and public protocol for easy access. However, because of this, it is also suceptible to cyber attacks such as DNS spoofing, which is when a hacker taps into a DNS server and changes it to match a domain name with a fake IP address. As the user, it is a big concern because you could be using that fake website to do financial transactions that do not go where they need to go. Thus, the ICANN plays an important role in making sure that the domain names entered in a web browser are going to the right IP addresses.
