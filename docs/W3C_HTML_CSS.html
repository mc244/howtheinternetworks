W3C, HTML, and CSS



The World Wide Web Consortium, or W3C, is an international association where its members work together to develop web standards and guidelines for the design of the web. It was founded in 1994 by the one of the most influential figures of computer science, Tim Berners-Lee. This is where the guidelines for using HTML5 and CSS came from. For the purpose of this project, HTML5 and CSS were both used in conjunction to create this website. In simple terms, HTML provides the structure for the basic layout and text on the website. CSS is used as a styling tool that polishes the HTML with cosmetic touch-ups. Both are necessary in order to create an aesthetically pleasing experience with clear readability when browsing the website.
